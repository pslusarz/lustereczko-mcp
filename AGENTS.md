# lustereczko-mcp

This is an [Model Context Protocol Application](https://apps.extensions.modelcontextprotocol.io/api/) extension to the MCP protocol. It is designed to be deployed locally, along the MCP host application and facilitate user's interaction in chat. 

## Stack

- **MCP server**: Python, FastMCP (`mcp/src/main/server.py`), use `uv` to run and build everything
- **Protocol**: MCP app extension (`text/html;profile=mcp-app`)

## Project Structure

```
mcp/src/main/    # MCP server source
skills/          # Skill definitions and recipes deployed to the server
docs/            # Documentation - offline, not deployed to the server
```

## API Reference

Use official reference whenever user asks specific question about the MCP apps protocol. Do not make assumptions, this documentation changes frequently, and the latest version is not in your training dataset.
[Official reference](https://apps.extensions.modelcontextprotocol.io/api/)

## Where we diverge from the official documentation
- We use Htmx as the basis for the UI (default documentation recommends other frameworks)
- Almost all of our UI is dynamically generated by the LLM during chat with the user. We do not have pre-made components

## Commands

```bash
# Run the MCP server
cd mcp && uv run python -m main.server
```
